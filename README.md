# Tramés — V1 Preview

A homepage redesign preview for **Tramés**, the AI-native OS for global supply chains.

**Live site:** https://niravbhatt1317.github.io/trames-v1-preview/

This is the *Structured Clarity* direction — polished, finance-grade, structured. Open it on desktop for the full experience (animated world map, scroll-driven sections, settings panel with theme + map style options in the bottom-right).

## Run locally

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Build

```bash
npm run build     # outputs to dist/
npm run preview   # serve the built output locally
```

## Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · framer-motion · d3-geo + topojson.
